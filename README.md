React를 활용하여 To do 웹 어플리케이션 만들기
기능
- 할 일 입력 및 삭제
- 날짜 및 시간 출력
- 배경화면 테마 변경

난이도
- 하
 
소요시간
- 3시간

배포
- 아마존 웹서비스 Amplify
